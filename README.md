# 内蒙古帅峰商贸公司网站

这是内蒙古帅峰商贸公司的官方网站，使用HTML、Tailwind CSS和JavaScript开发。

## 项目结构

```
├── index.html          # 网站首页
├── README.md           # 项目说明文件
└── .gitignore          # Git忽略文件
```

## 功能特点

- 响应式设计，适配不同屏幕尺寸
- 导航栏固定在顶部，方便用户导航
- 首屏英雄区，展示公司核心信息
- 关于我们部分，介绍公司历史和发展历程
- 产品展示部分，展示公司的业务服务
- 联系我们部分，提供在线留言功能
- AI智能客服，提供即时在线咨询

## 技术栈

- HTML5
- Tailwind CSS v3
- JavaScript
- Font Awesome
- Three.js (用于背景效果)

## 如何使用

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/jack-1151112367/sfsm.git
   ```

2. 打开 `index.html` 文件即可查看网站

## 部署

该网站可以直接部署到GitHub Pages：

1. 登录GitHub账号
2. 进入仓库设置
3. 在GitHub Pages部分，选择 `main` 分支作为源
4. 点击保存，GitHub会自动部署网站
